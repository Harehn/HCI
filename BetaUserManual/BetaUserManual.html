User manual

Project Proposal

We found a certain lack of addressing accessibility concerns from a subset of university websites. These issues extended to a lot of different websites in general. We focused on how to address the needs of color blind people in that aspect. We found one of the ways that colorblind people were impeded were the color schemes used in websites. We want to fix it using an chrome extension that allows users to change the colors on the screen by applying a filter over the website.

Introduction

Our product’s main feature is the ability to use saturation, presets and contrast of colors on the websites to allow colorblind people to better distinguish between the colors.

Installation

  1. Go to this link to download the zip file. (https://github.com/Harehn/HCI/blob/gh-pages/colorblind-extension.zip)
  2. Extract the zip file.
  3. Go to your chrome browser
  4. In the address bar, type in “chrome://extensions/.” and press Enter. You should see something similar to the illustration below
  5. Enable "Developer mode" in the top right.
  6. Click "Load unpacked" and select your extension folder.
  1. The folder should be the one containing the manifest.json File

Using the Product

  1. Go to the extensions of your google chrome
  1. You may choose to pin the extension for easier access
  1. Click on the extension.
  2. Click on apply colors.
